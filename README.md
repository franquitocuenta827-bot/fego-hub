# FEGO HUB

Roblox trading utility script.

## Features

- **FREEZE** - Freezes opponent's trade animation and movement
- **DISAPPEAR** - Makes opponent's character invisible in trade viewport

## Usage

1. Copy the contents of `FegoHub.luau`
2. Paste into your Roblox script executor
3. Execute the script
4. Toggle the GUI with **Right Control**

## Requirements

- Roblox script executor
- Must be in a trade or similar scenario with a visible opponent viewport
